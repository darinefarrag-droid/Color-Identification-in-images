# Color-Identification-in-images

An interactive color identification system that detects colors in images using RGB values. Users can upload an image and click on any pixel to get the color name along with its RGB components.

##  Overview
This project is an interactive color detection system that allows users to identify colors in any image.

##  How it works
The system reads pixel values from the image and matches them with a dataset of predefined colors.

##  Features
- Upload any image
- Click to detect color
- Show RGB values

##  Technologies Used
- Python
- OpenCV
- Pandas
- Tkinter

##  How to Run
1. Run the script
2. Select image
3. Click on image

## 📂 Dataset
Uses CSV file with color names and RGB values
